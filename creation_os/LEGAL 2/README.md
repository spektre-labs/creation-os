# Legal headers — Creation OS

## Canonical source-file header

Copy the **entire comment block** from [`SOURCE_FILE_HEADER.c`](SOURCE_FILE_HEADER.c) to the top of new Spektre-authored C, C++, or Objective-C++ sources. For **Python**, use [`SOURCE_FILE_HEADER.py`](SOURCE_FILE_HEADER.py) as the module docstring / top-of-file block.

- Keep **`SPDX-License-Identifier: AGPL-3.0-or-later`** as shown (first licence line) for SPDX / REUSE tooling.
- Adjust the line **“CREATION OS v2.0 — Core Source”** if the file is clearly auxiliary (e.g. “— Native M4 bridge”).
